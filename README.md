# Pimantel!

[![Daily German Puzzle](https://github.com/ronjang/pimantel/actions/workflows/daily-puzzle.yml/badge.svg)](https://github.com/ronjang/pimantel/actions/workflows/daily-puzzle.yml)
[![Pimantel des Tages](https://img.shields.io/endpoint?url=https://raw.githubusercontent.com/ronjang/pimantel/master/.github/badge/daily.json)](https://pimantel.de)

It's like Pimantle, but in German!

> This repo is a customized fork of Pimantle, adapted to use **German words**.
> Semantic similarity is powered by the
> [devmount German word2vec model](https://devmount.github.io/GermanWordEmbeddings/)
> (608k vocabulary, 300 dimensions). A daily GitHub Action generates a new
> puzzle and updates the badge above with the current puzzle number.

Play it at **[pimantel.de](https://pimantel.de)**.
